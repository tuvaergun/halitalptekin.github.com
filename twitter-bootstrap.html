Twitter Bootstrap

Jun 05, 2012

Benimde kişisel blogumda kullandığım arayuz tasarımı olan Bootstrap,duşunulduğunden daha da fazla kolaylıklar sağlıyor.Tabi teknik olarak kolaylıkların yanında zamandan ve paranızdan kar ediyorsunuz.Eğer sizde surekli olarak web siteleri geliştirip duruyorsanız Bootstrap sizin tasarım için harcadığınız zamanı gerçekten gozle gorulur derece de azaltacaktır.Bunun yanında gerçekten de harika bir gorselliğe sahip.Tabi guzellik goreceli bir kavram olabilir ama bana gore Bootstrap her kesime hitap eden bir tasarıma sahip.

Gidip birçok HTML/CSS tasarımına para vermek istemiyorsanız veya ucretsiz tasarımları beğenmediyseniz Bootstrap tam size gore.Kullanımı ve ozelleştirilmeside gerçekten çok kolay.Siz yeterki kullanın diye her turlu olanağı sağlamışlar.İçerisine de birçok eklenti eklemişler.Genelde surekli olarka kullanılan Javascriptleri de içerisine eklenti olarak eklemişler.Bunlarıda isteğinize gore kullanabiliyorsunuz.

Eğer sizde bu tasarım altyapısını merak ediyorsanız buradan resmi sitesine gidebilirsiniz.İndirmek içinde buradaki linki kullanabilirsiniz.

Eğer Bootstrap kullanmaya karar verirseniz aşağıdaki kaynaklara da bakmanızda fayda var.

Bootstrap ile geliştirilen uygulamalar ve kaynaklar

Jquery UI Bootstrap - Bootstrap ile Jquery'nın gucunu kullanan kit.

Bootstrap Image Gallery - Bootstrap için resim galerisi.

Jquery Mobile Bootstrap Theme - Jquery ve Bootstrap ile geliştirilmiş mobil tema.

Colorpicker and Detapicker - Bootstrap için renk ve tarih seçici.

StyleBootstrap - Birbirinden guzel bootstrap stilleri.

Beatiful Buttons - Guzel bootstrap butonları.

Bootswatch - Farklı renklerde guzel bootstrap tasarımları.

Fbootstrap - Facebook uygulamaları için bootstrap.

Bootstrap PSD - Bootstrap uygulamasının PSD hali.

Bootstrap Fireworks - Bootstrap ile geliştirilmiş guzel bir çalışma.Bakmanızı oneririm.

Django Bootstrap Toolkit - Django uzerinde bootstrap kullanmanıza yarayan uygulama.
