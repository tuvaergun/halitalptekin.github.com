Ubuntu

Jun 03, 2012 by halit in linux tagged linux ubuntu ubuntu ayarları ubuntu programlar

Buyuk bir değişiklik yapıp Windowstan Ubuntuya geçtiniz.Ancak daha ısınamadınız.Çunku birçok şey eksik.Bazı şeyler yavaş.Bugun sizlere ubuntu konusunda kesin bilmeniz gereken bazı konulardan bahsedeceğim.Unutmayın linux ve açık kaynak kod dunyası o kadar geniş ki emin olun ne isterseniz fazlasıyla var.Eğer bir konuda umitsizliğe duştuğunuzde direk windowsa geçmeyi duşunmeyin.Biraz araştırmayla daha onceden aynı sorunla karşılaşmış insanların yazılarını bulabilirsiniz.Açık kaynak kod dunyasına hiçbirşey demiyorum zaten.Uçsuz bucaksız bir dunya.

1-) Depoları Etkinleştirin

Ubuntu her yeni kurduğumda yaptığım ilk işlerden birisi depoların tumunu açmaktır.Sizde boyle yaparak daha fazla programa ulaşabilirsiniz. Bunun için sırayla; System>Administration>Synaptic Package Manager Girin. Ardından Settings>Repositories kısmına tıklayın.Açılan pencerede tum kısımları işaretleyin.Son olarakta Third-Party Software Penceresine gidip orada da tum seçenekleri işaretleyiniz.Pencereleri kapayıp Reload tuşuna basarak depoları guncelleyebilirsiniz.

2-) Grub Menu Editleme

Eğer sizde her açılışta Grub Menu'nun o sıkıcı siyah ekranından sıkıldıysanız bu madde tam size gore.Bunun yanında Ubuntunun yanında başka işletim sistemide varsa Grub menuden çekeceğiniz vardır.Bu sorunları halletmek için Startup Manager tam size gore.

sudo apt-get install startupmanager

Bu komut ile kuruyoruz.Kurduktan sonra çalıştırman hemen once grub dosyanızın yedeğini almakta fayda var.

sudo cp /boot/grub/menu.lst /yedek/yedek.lst

Program kurulduktan sonra Grub menuyu zevkinize gore duzenleyebilirsiniz.Zamanında işime çok yaramıştı.

3-) Guvenlik Duvarı

Aslında linux guvenlik olarak tatmin edici de olsa.Genede eşeğinizi sağlam kazığa bağlayın.Sonra başınız yanmasın.Ubuntu için bu işi iyi yapan GUFW var.Sizde eğer lazım goruyorsanız kurabilirsiniz.

sudo apt-get install gufw

Bunun yanında alternatif olarak aşağıdaki firewallı da kullanabilirsiniz.

sudo apt-get install firestarter

4-) Wine

Geldik linux için bugune kadarki belkide en yararlı yazılıma.Wine bir windows emulatorudur.Kısacası sanal bir windows ortamı yaparak .exe gibi windowsa ozgu dosyaları açmanızı sağlar.Her linux pcsinde bulunması gerekir.İlerde işiniz duşebilir.

sudo apt-get install wine

5-) Gerekli Ekstralar

Çok gerekli hatta zorunludur.Kurduğunuz her Linuxa ilk bunu yapın desem yalan olmaz.Flash Player,Java,Codecler vs hepsi bunun içindedir.

sudo apt-get install ubuntu-restricted-extras

6-) Ubuntu Tweak

Ubuntu için gorsel duzenlemeler,sistem ayarları gibi gerekli konularda değişiklik yapmamıza olanak sağlayan programdır. Kurulum için once

gksu gedit /etc/apt/sources.list

Komutunu çalıştırıp bu dosyanın içine aşağıdaki kodları yapıştırın.

deb http://ppa.launchpad.net/tualatrix/ubuntu jaunty main 
deb-src http://ppa.launchpad.net/tualatrix/ubuntu jaunty main

Son olarakta aşağıdaki komutlarla depoları guncelleyip ubuntu tweakı kurun.

sudo apt-get update


 sudo apt-get install ubuntu-tweak

7-) VLC

Duymayan varmıdır bilmemde Linux platformların vazgeçilmez Media Playerıdır.Bu yuzden kurmakta fayda var.

sudo apt-get install vlc

8-) Gnome Do

Çok hoşuma giden bir uygulamadır.Benim onerim kesin kurun.

sudo apt-get install gnome-do
