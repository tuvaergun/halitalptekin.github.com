C Programlama Dili Kaynaklari

Jun 03, 2012

Programlama sadece bilgisayar bilimleri ile uğraşanların değil matematik,fizik,astronomi gibi alanlar ile uğraşanların da vazgeçilmezidir.Ve birçok universitede programlamaya C dili ile başlıyor.C programlama dili ile aklınıza gelecek neredeyse herşeyi yapabilirsiniz.Sonuçta işletim sistemleride dahil olmak uzere hala kullandığımız birçok program,programlama dili onunla yazıldı.Emin olun yazılmaya da devam edecektir.

Çunku C ile yeni birşeyler yapmak için biraz C bilgisi ve hayal gucu yeterli oluyor.Bunun yanında karşılaştığınız birçok problemi de onunla çozebiliyorsunuz.Eğer sizde bu bu dile başlamak için kaynak arıyorsanız doğru yerdesiniz.Bana gore piyasada bulabileceğiniz en kaliteli ilk 10 kaynak aşağıdadır.Tabi bunlar ucretsiz olanları.Birde ucretli olanları var.Onlarıda ilerleyen zamanlarda paylaşabilirim.

1-) C Programlama Notları

2-) Programming in C - UNIX System Calls and Subroutines using C

3-) C Lesson by Chris Sawtell

4-) comp.lang.c Frequently Asked Questions

5-) Collection of C programming material at Lysator Society

6-) A collection of useful libraries written in C

7-) The C Programming Language Standard

8-) High-level and Low-level Programming in C

9-) Microsoft C Language Reference

10-) Online C/C++ Tutorials and Documentation

Biraz da gorsel kaynak arıyorsanız Buradaki yazıma bakmanızı oneririm.
