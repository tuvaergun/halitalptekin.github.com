Webmasterlar İçin Jquery Tools

 Jun 02, 2012 by halit in programlama tagged jquery jquery örnekleri jquery tools jquery uygulamaları

Jquery çıktığı ilk gunden bugune hala çok populer.Birçok webmaster Jqueryden faydalanıyor.Benim siteminde birçok kısmında bu kodlardan var.Bende bugun sizlere jquerynin en işlevsel arayuzu bileşenlerinden oluşan Jquery Tools koleksiyonunu paylaşıcam.Tarih seçiciler (datepicker), ipucu balonları (tooltips), form kontrolleri, tablar gibi daha onlarca işe yarar ornek detaylı anlatımlarıyla birlikte koleksiyonda mevcut.Arşivinizde bulunmasında fayda var.

Site Anasayfası: http://flowplayer.org/tools/index.html

Örnek Uygulamalar: http://flowplayer.org/tools/demos/index.html

İndirme Linki: http://flowplayer.org/tools/download/index.html
