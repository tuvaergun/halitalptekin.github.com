Ücretsiz Bulut Hosting

 Jun 03, 2012 by halit in internet tagged bulut bilişim bulut hosting cloud hosting ücretsiz hosting

Son gunlerde artık çok fazla duymaya başladığımız bir kelime bulut bilgisayarlar.Turkçeye çevrilmiş haliyle çok garip duruyor ama zamanla alışacağız.Yabancıların Cloud Computing dediği bu sistem bir bilgisayarın kaynaklarının sanallaştırma yazılımlarıyla birçok kaynağa paylaştırılması anlamına geliyor.Eğer bu sistemi daha yakından tanımak için ucretsiz hosting aradıysanız husranla karşılaşmışınızdır.Bu yuzden sizlere bu alandaki en iyi site olan dotcloud sitesinden bahsedeceğim.

Şu anda 19 adet servisi destekleyen dotcloud bazı kısıtlamalar dışında paralı birçok hostingten daha kaliteli bir hizmet veriyor.Bu servislerle çok unlu wordpress,wiki vb gibi sistemleride tam performansla çalıştırabiliyorsunuz.Dotcloud sistemini kullanan ornek sitelere buradan ulaşabilirsiniz.

Dotcloud’un fiyatlandırma politikasıda saniyede yaptığınız işlem başına gore değişiyor.Eğer saniyede 1000 işlem yapan çok buyuk bir siteniz yoksa dotcloud tum işleriniz için yeterli olacaktır.

Bu sistemi kullanmak için once buradan kayıt olun.Sonra ayarlar sayfasından api anahtarınızı bir yere not alın.

Bundan sonra Linux sisteminizde dotcloud clienti kurucağız.Bunun için once pip paketini kuruyoruz. 

sudo easy_install pip

Ardından dotcloud’u kuruyoruz. 

sudo pip install dotcloud

Bunları tamamladıktan sonra artık dotcloud pcmize kuruldu.Şimdi terminalden dotcloud diyerek programa giriş yapın.Önceden not aldığınız api keyi burada gereken yere yazın. Artık kurulum bitti.Bundan sonra istediğiniz sistemi kullanmak için proje oluşturacağız. 

dotcloud create halitalptekin

Kullanmak istediğiniz servisi dotcloud.yml dosyasında tanımlıyorsunuz.Aşağıda sadece bir ornek vardır.Ve eksiktir.Aşağıdaki dokumanları okumadan servera upload etmeyiniz. 

www:
  type : python
db:
  type : postgresql

Burada Python ile postegresql servisini kullanıyoruz. Bununla ilgili sitenin ayrıntılı dokumanlarına [buradan][3] ulaşabilirsiniz. Projeniz bitincede aşağıdaki komut ile siteye upload ediyoruz. 

   [3]: http://docs.dotcloud.com/services/

dotcloud push halitalptekin

Cloud computing hakkında detaylı bilgi için buraya,Turkçe kaynak İçin buraya bakabilirsiniz.
